# 🛒 Grocery Website - Frontend

This is the frontend of a Grocery E-commerce Web Application built using Next.js. It provides a fast, modern, and responsive user interface for browsing products, managing cart items, and placing orders online.

---

## 🚀 Features

* 🥦 Browse grocery products by category
* 🔍 Search and filter products
* 🛒 Add to cart & manage cart items
* 💳 Seamless checkout experience
* 📱 Fully responsive design
* ⚡ Fast performance with Next.js

---
## 🛠️ Tech Stack

* Next.js
* React.js
* Tailwind CSS / CSS Modules
* Axios (for API calls)

---

## 📂 Folder Structure

```
/components   → Reusable UI components  
/pages        → Application pages  
/services     → API calls  
/styles       → Styling files  
/public       → Images & assets  
```

---

## ⚙️ Setup Instructions

### 1️⃣ Clone the Repository

```
git clone https://github.com/RUSHANG3003/grocery-frontend.git
cd grocery-frontend
```

### 2️⃣ Install Dependencies

```
npm install
```

---

### 3️⃣ Environment Variables

Create a `.env.local` file in the root:

```
NEXT_PUBLIC_API_URL=http://localhost:3000/api
NEXT_PUBLIC_RAZORPAY_KEY=your_key
```

---

### 4️⃣ Run the Project

```
npm run dev
```

App will run on:
👉 http://localhost:3000

---

## 🔗 Backend API

Make sure your backend server is running before using the frontend.

Example:

```
http://localhost:3000/api
```
